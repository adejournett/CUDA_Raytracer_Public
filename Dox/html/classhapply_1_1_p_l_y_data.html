Common-case helper to get face indices for a mesh.

If not template type is given, size_t is used. Naively converts to requested signedness, which may lead to unexpected values if an unsigned type is used and file contains negative values.

Returns
The indices into the vertex elements for each face. Usually 0-based, though there are no formal rules.
