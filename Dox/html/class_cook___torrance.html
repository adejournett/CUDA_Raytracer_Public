Detailed Description

Cook-Torrance material This implementation is based on the paper "A Reflectance Model for computer graphics" Simulates roughness on the 1 micron scale via probability distribution functions

All matghematics can be found in Cook-Torrance. The distribution can be found in Foley et. al. Computer Graphics: Principles and Practice 3rd ed.
